ColorEcho
=================

ColorEcho is a CLI tool for PHP to echo colored string.

Quick start
-----------
	require 'ColorString.php';
	require 'ColorEcho.php';

	ColorEcho::info('Hello, world!');
	echo PHP_EOL;
	ColorEcho::warn('Hello, world!');
	echo PHP_EOL;
	ColorEcho::error('Hello, world!');
	echo PHP_EOL;
